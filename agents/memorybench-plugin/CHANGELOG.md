# Changelog

All notable changes to the MemoryBench Plugin will be documented in this file.

## [1.0.0] - 2024-11-29

### Added
- Initial release of MemoryBench Core Plugin
- Memory Organizer Agent - auto-categorization and deduplication
- Memory Retriever Agent - semantic search and retrieval
- Memory Summarizer Agent - digests and insights
- Memory Parser Skill - entity extraction
- Memory Linker Skill - relationship discovery
- Pre/post save hooks for memory processing
- Query enhancement for better search
- Status and Logs commands
- MCP server integration for indexing
