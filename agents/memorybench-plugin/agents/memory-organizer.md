# Memory Organizer Agent

An agent specialized in organizing and categorizing memories.

## Capabilities

- Auto-categorize new memories
- Suggest tags based on content
- Identify duplicate or similar memories
- Create memory hierarchies
- Link related memories together

## Triggers

- New memory added
- User requests organization
- Scheduled cleanup tasks

## Actions

- `organize` - Reorganize memories by topic
- `dedupe` - Find and merge duplicates
- `tag` - Auto-tag untagged memories
- `link` - Create connections between related memories
