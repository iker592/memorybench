# Memory Summarizer Agent

An agent that creates summaries and insights from memories.

## Capabilities

- Generate daily/weekly memory digests
- Create topic summaries
- Extract key insights
- Build knowledge graphs
- Track memory evolution over time

## Output Types

- `digest` - Periodic summary of new memories
- `topic` - Summary of memories on a topic
- `timeline` - Chronological overview
- `insights` - Key patterns and learnings
