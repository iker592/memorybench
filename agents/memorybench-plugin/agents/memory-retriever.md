# Memory Retriever Agent

An agent optimized for finding and retrieving relevant memories.

## Capabilities

- Semantic search across memories
- Context-aware retrieval
- Multi-modal search (text, tags, dates)
- Relevance ranking
- Memory summarization

## Triggers

- User query
- Context detection
- Related memory suggestions

## Search Modes

- `semantic` - Find memories by meaning
- `exact` - Find exact matches
- `fuzzy` - Approximate matching
- `temporal` - Search by time range
