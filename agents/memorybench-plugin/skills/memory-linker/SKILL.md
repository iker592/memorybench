# Memory Linker Skill

A skill for discovering and creating connections between memories.

## Description

This skill enables the agent to:
- Find related memories
- Create bidirectional links
- Build knowledge graphs
- Suggest memory connections
- Visualize memory relationships

## Input

- Source memory ID or content
- Link type (related, parent, child, reference)

## Output

- Related memory IDs
- Relationship strength scores
- Suggested link descriptions
