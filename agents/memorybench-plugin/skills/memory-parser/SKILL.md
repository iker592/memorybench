# Memory Parser Skill

A skill for parsing and extracting structured data from unstructured memories.

## Description

This skill enables the agent to:
- Extract entities (people, places, dates)
- Identify key concepts
- Parse structured data from text
- Detect memory type (note, idea, task, reference)

## Input

- Raw memory content
- Optional parsing hints

## Output

- Extracted entities
- Key concepts
- Suggested category
- Structured metadata
